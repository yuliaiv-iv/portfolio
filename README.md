## Portfolio

Ссылка на **[GitHub pages](https://yuliaiv-iv.github.io/portfolio/)**
### Технологии

* **`React`**
* **`React router dom`**
* **`HTML5, CSS3`**
* **`Адаптивная верстка`** с использованием **`медиа-запросов`**
* **`Flex`** и **`Grid-layout`**
* **`Styled components`**
* **`Framer Motion library`** реализация анимации

### Запуск и установка
* `npm install`
* `npm start`
